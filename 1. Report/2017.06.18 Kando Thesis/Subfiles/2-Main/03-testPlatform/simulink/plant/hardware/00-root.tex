The \tif{Hardware} variant of the plant contains subsystems for the various hardware components,
{\fns[\tif{not including the controller}]}.
It is depicted in Figure~%
\ref{FIG:testPlatform:simulink:plant:hardware}.

The hardware components include:

\begin{itemize}[leftmargin=*,itemsep=-1em]

\item Clock
      
\item Gyroscope
      
\item Accelerometer
      
\item Motor Drivers
      
\item Motor Encoders

\end{itemize}
